# Commit

## Realizar um commit

## Alterar a mensagem do último commit

```bash
git commit --amend -m "new message"
```

Outra forma (neste caso, vai abrir o editor padrão do sistema):

```bash
git commit --amend
```

Se usa editor vin:

- apertar `i` para insert.
- apertar `ESC` + `:wq` para salvar e sair.
